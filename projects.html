Combat cowboy is a 1v1 multiplayer online game that I made as a school project in my junior year of highschool. It used python and the pygame library. Play audio file to the left to hear the background music used in the game.

Target Clicker

2020

Target clicker is a python based web game I created during the covid pandemic in order to teach myself python. It uses a clock to calculate the amount of time it takes a user to complete the short game. After completion, the time is compared to the daily record and if beaten a prompt appears to change the name of the daily record on scoreboard. (Click the target to play)
